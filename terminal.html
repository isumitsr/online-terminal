>> The secure, jailed, stand-alone linux environment to test code online.

Instructions:

    * Use Linux commands to edit, compile & run the code.
    * Code from the text area is saved to  main.cpp 
    *  main.cpp  is compiled to  myprog  by default
    * Use  nano  or  vim  to edit file.
      Eg.  nano main.cpp 
             ctrl + x  to write changes made in the file.
            Press  y  to save the file.
    * Compile it manually:  c++ main.cpp -o myprog 
    * Execute:  ./myprog 
    * Use  howto  command inside the terminal to display 
      these instructions again.
    * C/C++ environment can be launched,
      support for other compilers will be available soon.

Enjoy!
